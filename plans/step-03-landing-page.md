# Step 3 — LandingPage: публичный список типов встреч

Гость видит список доступных типов встреч.

## Работа

- `LandingPage` переписывается:
  - Загружает `GET /event-types`.
  - Показывает карточки (название, описание, длительность).
  - Кнопка «Забронировать» ведёт на `/book?eventTypeId=<id>` (опциональный предвыбор).
- Пустое состояние, когда типов нет.

## TypeSpec / БД

Без изменений.

## Тесты

Компонентные тесты рендера списка и пустого состояния (с моком клиента).

## Состояние после шага

`/` показывает актуальный список типов встреч из БД.
