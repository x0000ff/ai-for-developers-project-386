# Step 7 — AdminPage: список предстоящих встреч

Владелец видит единый список предстоящих встреч.

## Работа

- На `AdminPage` добавляется секция «Предстоящие встречи»:
  - Таблица с `startsAt`, `endsAt`, `eventTypeName` (или пометка «Тип удалён», если `eventTypeId === null`), `guestName`, `guestEmail`.
  - Данные из `GET /bookings`.
  - Автообновление кнопкой «Обновить» (polling не делаем).
  - Пустое состояние.
